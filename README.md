# KeyRun

**KeyRun** is a small game created during the **Semester 2 Game Jam** organized by the **Game Development Guild (GDG)**.  
The jam theme was **"Pass the Torch"**, and the game was built around the idea of transferring control between the runners during gameplay.

## Concept

In KeyRun, players run while coordinating control to pass on to the next coming player
At certain moments, the torch must be passed to the other player by selecting a number input

## Game Jam Modifier

To satisfy the game jam modifier, the game was designed to use a MIDI keyboard as the primary input device.

Specific MIDI keys were mapped to **move the character up and down**

When approaching a torch pass point, players select **1–3** on the MIDI keyboard to **pass the torch to the other player**

## Current Controls (Keyboard)

Keyboard controls were not implemented during the jam, but basic keyboard support has now been added:

- **↑ / ↓ Arrow Keys** – Move up and down
- **1 / 2 / 3** – Pass the torch when you are near the pass point
